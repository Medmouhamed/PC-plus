# 💻 PC-Plus | Full-Stack Computer Store

A high-performance E-commerce platform dedicated to PC hardware, built with the **MERN** logic (React, Node.js, Express, MongoDB). This project demonstrates a complete integration between a modern **React 19** frontend and a robust **RESTful API**.

## 🏗️ Architecture Overview
The project is structured as a decoupled application:
* **Frontend**: Built with **React 19** & **Vite** for a seamless user experience.
* **Backend**: A **Node.js** & **Express** server located in `/pcstore-backend-main` handling business logic and database operations.

## 🚀 Technical Features

### 🔹 Frontend (Client-Side)
* **Latest React Hooks**: Utilizing React 19 for efficient state management.
* **Vite 7**: Ultra-fast build tool and development server.
* **React Router 7**: Dynamic routing for product categories and details.
* **React Icons**: Professional UI components and iconography.

### 🔹 Backend (Server-Side)
* **Express.js Framework**: Handling RESTful API endpoints for products and users.
* **Database Integration**: Connectivity with **MongoDB** for flexible hardware data storage.
* **Middleware**: Security and request parsing optimized for e-commerce traffic.
* **Scalable Structure**: Organized routes and controllers for easy maintenance.
